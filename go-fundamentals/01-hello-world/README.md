# 01: Hello, world

First Go program using TDD.

## Files

- `hello_test.go` - Test for Hello function
- `hello.go` - Hello function implementation
- `go.mod` - Module file (like package.json)

## Run

```bash
go test
```

## Concepts

- TDD workflow
- Basic Go syntax
- Function declaration
- Testing with `testing` package

## Go Workflow

```bash
go mod init project-name  # Setup
go test                  # Run tests
```
